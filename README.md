# RainMusic
RainMusic是一个使用Jetpack Compose构建的网易云第三方app, 采用Material You设计，专注听歌功能，没有社交功能，还你一个纯净的音乐APP

## 注意事项
1. 本APP完全免费，请勿用于商业用途，仅供学习个人使用
2. 请勿提交任何"破解VIP", "破解灰色歌单" 之类的功能请求或者PR, 不会添加任何侵权功能

## 已支持的功能
* 无

## 计划支持的功能
* 歌单
* 私人FM
* 搜索
* 每日推荐
* 歌词

## 下载
* 开发中，暂时无法下载

## 技术栈
* Jetpack Compose
* Kotlin Flow / Coroutine
* Navigation
* Room
* Retrofit
* Hilt
